# 💰 Finanças Pessoais Inteligentes

Uma aplicação web moderna e intuitiva desenvolvida para ajudar no controle financeiro diário, organização de despesas e planejamento de metas. O projeto combina inteligência visual e facilidade de uso para transformar a relação das pessoas com o dinheiro.

## ✨ Principais Funcionalidades

* 📊 **Dashboard Dinâmico:** Visualização clara de receitas, despesas e saldo atual por meio de gráficos intuitivos.
* 🏷️ **Categorização de Gastos:** Separação de despesas por categorias (Alimentação, Lazer, Transporte, Moradia) para identificar para onde o dinheiro está indo.
* 🎯 **Metas de Economia:** Definição e acompanhamento de objetivos financeiros a curto e longo prazo.
* 📱 **Design Responsivo:** Interface totalmente adaptável para smartphones, tablets e computadores.

---

## 🛠️ Tecnologias e Ferramentas

O desenvolvimento foi guiado por inteligência artificial através da plataforma **Lovable**, garantindo um código limpo, componentizado e focado na experiência do usuário ($UX$).

* ⚛️ **React** — Biblioteca para construção da interface de usuário.
* 📘 **TypeScript** — Tipagem estática para maior segurança e escalabilidade do código.
* 🎨 **Tailwind CSS** — Framework utilitário para estilização rápida e design moderno.
* 🤖 **Lovable AI** — Plataforma utilizada para a concepção ágil do MVP (Produto Mínimo Viável).

## 📂 Como testar localmente

Se desejar rodar o projeto na sua máquina:

1. Clone o repositório principal.
2. Navegue até a pasta do projeto.
3. Instale as dependências com `npm install`.
4. Inicie o servidor de desenvolvimento com `npm run dev`.
